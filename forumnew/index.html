The RECON forum

The RECON forum provides a platform for public as well as private discussions relating to the use and development of tools for disease outbreaks analysis in R. Our forum is currently hosted at Vanilla forums.


How to use the forum?

To use the forum, you will need to create an account on Vanilla forums. This is particularly easy, as you can use various credentials to log in (e.g. Google, Twitter, LinkedIn, Facebook).
